# Challenge 1 - Pandas 📊

Este proyecto corresponde al Challenge del curso de **Pandas** de Alura Latam.

El objetivo principal consiste en analizar los datos de cuatro tiendas pertenecientes al señor Juan, con el propósito de determinar cuál de ellas presenta el menor rendimiento y podría ser considerada para su venta. Para ello, se realiza un análisis exploratorio de datos que permite evaluar distintos indicadores de desempeño comercial.

## 📌 Contenido del análisis

1. **Importación de datos**
   Se cargan los datasets correspondientes a cada tienda para su posterior análisis.

2. **Análisis por tienda**
   Se evalúan métricas relevantes como:

   * Ingresos totales
   * Ventas por categoría
   * Calificación de clientes
   * Distribución geográfica de las ventas

3. **Visualización de datos**
   Se generan gráficos que facilitan la interpretación de la información, tales como:

   * Productos más y menos vendidos
   * Comparación de ingresos entre tiendas
   * Mapas de dispersión y mapas de calor (heatmaps) de ventas

4. **Informe final**
   Se presentan los principales hallazgos del análisis junto con recomendaciones basadas en los datos obtenidos.

## ⚙️ Instrucciones de uso

Para ejecutar el proyecto:

1. Descargar el archivo `.ipynb`.
2. Subirlo a **Google Colab**.
3. Ejecutar todas las celdas en orden.
